## Build Setup
Clone the repo

```
https://github.com/a-meti/russian-travel.git
```
Install dependencies
```
npm install
```
Server with hot reload at http://localhost:9000/
```
npm run start
```
Production Build
```
npm run build
```
Development mode
```
npm run dev
```
